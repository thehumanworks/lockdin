# LockdIn Chrome Web Store Draft

## Store title

LockdIn

## Summary

Hide LinkedIn's feed, news, and notifications to make space for focused work.

## Description

LockdIn turns LinkedIn's home experience into a quieter workspace.

When enabled, it:

- hides the main feed
- hides the LinkedIn News rail
- hides the Notifications nav item
- hides the extra right-side scaffold rail that can appear in Messaging
- keeps the profile rail on the left
- replaces the rest of the page with a full-width motivational backdrop

LockdIn is designed for people who want to use LinkedIn intentionally without getting pulled into the default scrolling loop.

What you get:

- a cleaner LinkedIn layout
- a rotating set of 50 motivational prompts
- bundled local assets with no remote runtime dependencies for the visual experience
- support for Chromium-based browsers that accept Chrome extensions

## Privacy disclosure draft

LockdIn does not collect, store, or transmit personal data.

The extension runs locally in your browser and only modifies the appearance of pages on `linkedin.com`.

It does not send browsing activity, account data, messages, or page content to any external server.
